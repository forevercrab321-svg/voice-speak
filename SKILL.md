# Voice Speak - 让 AI Agent 语音讲话

让 OpenClaw 可以语音说话提醒主人，并通过 Telegram 发送语音消息。

> ⚡ 基于 OpenAI TTS，声音自然流畅！

## 特色

- 🎤 **自然语音** - 使用 OpenAI TTS，声音清晰自然
- 🦞 **萌系风格** - 默认使用活泼可爱的声线
- 📱 **Telegram 集成** - 支持发送语音到 Telegram
- 🔔 **智能提醒** - Agent 可主动触发语音提醒

## 工具

### tts - 语音播放/发送

把文字转换成语音并播放或发送。

**参数:**
- `text` (必填): 要说的话
- `channel` (可选): 发送频道
  - `telegram` - 发送到 Telegram
  - 不填则只本地播放

**使用示例:**
```
tts: 你有新的邮件
tts: 主人，该休息了
tts(text=今天天气不错, channel=telegram)
```

## 声音风格

推荐使用以下语音风格:
- `shimmer` - 温柔可爱的女声 ⭐推荐
- `nova` - 活泼女声
- `fable` - 故事感男声
- `onyx` - 成熟男声
- `alloy` - 中性声音

## 触发场景

Agent 可以主动触发语音提醒:
- 📧 重要通知 (邮件、会议、消息)
- ✅ 需要确认 (决策需要主人同意)
- 🚨 紧急情况 (系统错误、安全警报)
- ⏰ 定时提醒 (喝水、运动、休息)
- 💕 情感交互 (早安、晚安、关心)

## 使用示例

### 本地语音提醒:
```
tts: 嘿Lee，你有一封新邮件
```

### 发送语音到 Telegram:
```
tts(text=会议将在10分钟后开始, channel=telegram)
```

### 萌系语气示例:
- "嘿主人~ 你有新的消息哦！"
- "休息时间到啦，要不要喝点水？"
- "晚安~ 祝你好梦！"

## 安装

1. 确保已配置 OPENAI_API_KEY
2. 直接使用内置的 `tts` 工具即可

## 配置

环境变量:
- `OPENAI_API_KEY` - OpenAI API Key (用于生成语音)

## 注意事项

- 本地播放会让周围的人都能听到
- Telegram 发送需要网络连接
- 建议设置触发规则避免过度打扰
